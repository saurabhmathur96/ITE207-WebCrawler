# ITE207-WebCrawler

Project for ITE207: Programming in Java.

Check Documentation directory for abstract

#### Requirements
  - Java 
  - Maven


####  Build Instructions
  1. `$ mvn install`
  2. `$ mvn compile`


#### To run the app
     `$ mvn exec:java -Dexec.args="config.xml"
     
#### config.xml
    - DB Url
    - Number of iterations
    - Seed Urls
    - Thread Pool Configurations
